# GitHub Pages 部署指南

## 常见问题及解决方案

### 1. 图片/资源加载问题
- 确保所有文件名都是小写（GitHub Pages 区分大小写）
- 使用相对路径（已在代码中实现）
- 检查文件是否已提交到 Git

### 2. 样式显示异常
- 确保 `style.css` 文件正确引用
- 清除浏览器缓存后重新加载
- 检查浏览器控制台是否有错误

### 3. 页面尺寸问题
- 该应用设计为固定尺寸（375×812px，模拟 iPhone X）
- 在不同尺寸屏幕上可能需要调整
- 可以修改 `style.css` 中的 `.app-container` 尺寸

## 部署步骤

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中开启 GitHub Pages
3. 选择 `main` 分支作为源
4. 等待几分钟后访问生成的 URL

## 本地测试

在本地打开 `index.html` 测试，确保一切正常后再部署。
